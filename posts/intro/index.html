위 영상은 안드로이드 구글 기술 영상이다. 이때 안드로이드 내부 기술에 대한 전반적인 내용이 나와있다.

Android Java Code From *.apk

이러한 기술은 세상에 알려지지 않은 Close-Code 를 연구하는 목적으로 사용한다. 해당 블로그 역시 그러한 관점으로 글을 포스팅할 예정이다. 해당 기술 결과물은 2차 사용 관련해서 합법적이지 않을 수 있어 주의가 필요하다.

flowchart LR
